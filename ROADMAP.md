# Roadmap

## Release Readiness

- Publish signed Linux AppImage and Windows installer artifacts.
- Add branded icons and installer artwork.
- Add more fixture-based Smart Director tests.
- Add public demo recordings made with synthetic sample data.

## Capture And Telemetry

- Harden GNOME and KWin compositor adapters across more distro versions.
- Expand Windows telemetry QA on real Windows 10/11 systems.
- Add opt-in native binary helpers for platforms where scripts are too slow or
  too limited.
- Add macOS capture and telemetry research after Linux/Windows stabilize.

## Editing And Rendering

- Add richer timeline editing for split/merge/drag zoom blocks.
- Add render regression tests with `ffprobe` output validation.
- Add more QA checks for text legibility, camera speed, and dropped-frame
  clusters.
